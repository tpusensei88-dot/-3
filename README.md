# なんじ なんふん アタック

小学校1年生向けの時計読み取り学習アプリです。

## セットアップ

```bash
npm install
npm run dev
```

## 使い方

1. **鍵開け問題**を最初にクリアしてください
2. クリア後、他のモードが解放されます
3. 各モードで10問の問題に挑戦します
4. 正答率とクリア時間に応じてスコアと称号が決まります

## モード説明

- **鍵開け問題**: 時計の針を動かして時刻を合わせます。リアルタイムで時刻が表示されます。
- **時計合わせモード★**: 鍵開け問題と同じですが、リアルタイム表示がありません。
- **何時何分か答えるモード★★**: 時計を見て時刻を答えます。分の数字が表示されます。
- **何時何分か答えるモード★★★**: 時計を見て時刻を答えます。分の数字は表示されません。

## 称号

- **ノーマル**: スコア 400未満
- **スーパー**: スコア 400以上
- **ハイパー**: スコア 700以上
- **マスター**: スコア 1000以上
